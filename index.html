Understanding the DevOps Lifecycle

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops) to shorten the systems development life cycle while delivering features, fixes, and updates frequently in close alignment with business objectives.

Plan

Define the features and capabilities of the application.

Code

Design, code, and build the application.

Build

Compile and package code for testing.

Test

Run automated tests to ensure quality.

Release

Approve and manage application versions.

Deploy

Schedule and implement changes to production.

Operate

Manage the production environment.

Monitor

Identify and track issues in production.
